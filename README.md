sabotage2_revamp
================

Another web forum, this time as a vanity project.

The .env file skeleton:

FLASK_CONFIG=production

DATABASE_URL=database://url

MEDIA=<path to media files>

MAIL_PREFIX=[haxorbb]

MAIL_REPLY=do-not-reply@example.com

MAIL_SERVER=localhost

MAIL_USE_TLS=0

MAIL_USE_SSL=0
